# using-GPT-API-to-make-translator 

api를 가져와서 사용하는 것을 연습해보기 위해 간단하게 만들어봤습니다 

### 사용한 패키지는 아래와 같습니다

### npm i ejs express body-parser openai dotenv

.env 파일에 OPENAI-API-KEY 값에 OpenAi에서 자신의 key를 만들어서 저장해주어야 실행됩니다
